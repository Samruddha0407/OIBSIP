# Buddy AI Landing Page

A modern and responsive AI landing page developed using HTML5, CSS3, and JavaScript. This project was created as part of the Oasis Infobyte Web Development & Designing Internship Program.

---

# Project Overview

Buddy AI is a modern landing page designed for an artificial intelligence company or startup. The website demonstrates front-end development concepts such as responsive layouts, clean UI design, dark/light theme functionality, and structured section-based navigation.

The main objective of this project is to build a professional and visually appealing landing page using core web development technologies.

---

# Features

## Navigation Bar
- Sticky navigation bar
- Smooth scrolling navigation
- Responsive menu layout
- Hover effects on navigation links

## Hero Section
- Attractive heading and description
- Centered call-to-action button
- Responsive AI-themed image

## Features Section
Includes feature cards for:
- AI Automation
- Smart Analytics
- Cloud Security

Each card contains:
- Icon
- Title
- Description
- Hover animation

## About Section
- Information about Buddy AI
- Responsive two-column layout
- Professional technology image

## Contact Section
- Contact email
- Social media icons
- Call-to-action area

## Dark and Light Mode
- Functional dark/light mode toggle button
- Dynamic theme switching
- Smooth transition effect

## Responsive Design
The website is fully responsive and works properly on:
- Desktop
- Tablet
- Mobile devices
